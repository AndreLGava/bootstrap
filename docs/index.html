Projetado para qualquer um, em qualquer lugar.

Bootstrap torna o desenvolvimento front-end web mais rápido e fácil. Ele é feito para pessoas de todos os níveis e dispositivos de qualquer forma ou tamanho.

Pré-processadores

Bootstrap usa CSS tradicional, mas seu código fonte utiliza os dois pré-processadores CSS mais populares, Less e Sass. Comece o desenvolvimento de forma rápida com seu CSS pré-compilado o construa com seu código fonte.

Um framework, qualquer dispositivo.

Bootstrap é eficiente e fácilmente adaptado para tamanhos de telas de seus websites e aplicações com seu simples código, para telefones, tablets, e desktos usando media queries CSS.

Muitos recursos

Com Bootstrap, você tem disponivel uma belissima documentação para elementos HTML, dezenas de componentes CSS e HTML, e incriveis plugins jQuery.

Bootstrap tem código aberto. É hospedado, desenvolvido e gerenciado no GitHub.
